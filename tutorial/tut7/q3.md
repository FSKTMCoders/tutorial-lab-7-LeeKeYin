3. Write a program that convert a sentence into binary number (ASCII code 8 bit) and store it in a text file named data.txt. Then, read from the text file and display the sentence.

```java
import java.io.*;
public class T7 {

    public static void main(String[] args) throws IOException {
        String sentence = "Hello World!";
        PrintWriter out = new PrintWriter("data.txt");
        for(char c : sentence.toCharArray()) {
            String binary = String.format("%8s", Integer.toBinaryString(c)).replace(' ', '0');
            out.println(binary);
        }
        out.close();
        
        BufferedReader in = new BufferedReader(new FileReader("data.txt"));
        StringBuilder decoded = new StringBuilder();
        String line;
        while((line = in.readLine()) != null) {
            int charCode = Integer.parseInt(line, 2);
            decoded.append((char) charCode);
        }
        in.close();
        System.out.println("Original: " + sentence);
        System.out.println("Decoded: " + decoded.toString());
    }
}
```
